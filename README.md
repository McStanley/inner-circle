## Tech stack

- TypeScript
- Express
- NodeJS
- MongoDB
- Tailwind
- Pug

## Prerequisites

- node >=20.10.0

## Install

```sh
npm install
```

## Usage

### Environment variables (`.env`)

```toml
MONGODB_URI="<mongodb_connection_string>"
SESSION_SECRET="<express-session_secret>"
```

### Run scripts

```sh
npm run build
npm run start
# OR
npm run dev
# OR
npm run debug
```
